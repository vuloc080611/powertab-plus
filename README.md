# PowerTab Plus — The All-in-One Chrome Extension You Can't Live Without

PowerTab Plus is a must-have productivity Chrome extension that supercharges your browsing experience.  
Manage sessions, switch tabs instantly, toggle smart dark mode, and keep quick notes — all from a beautifully designed popup and side panel.

## ✨ Features

- 📁 **Session Manager** — Save all open tabs as a session and restore them anytime. Perfect for research, work sprints, or context switching.
- 🔖 **Tab Switcher & Search** — Search across all open tabs by title or URL and jump to the one you need in one click.
- 🌓 **Smart Dark Mode** — Toggle dark mode for any website. Your preference is remembered per domain and automatically reapplied.
- 📝 **Quick Notes** — Write and auto‑save notes that sync across all tabs. Access them from the popup or the dedicated side panel.
- ⌨️ **Keyboard Shortcuts** — Boost your workflow with global shortcuts:
  - `Ctrl+Shift+S` — Save current session
  - `Ctrl+Shift+D` — Toggle dark mode for the active tab
  - `Ctrl+Shift+E` — Open extension popup
- 🧩 **Side Panel** — Open PowerTab Plus in Chrome's side panel for persistent access to notes and tabs (Chrome 114+).
- 🎨 **Clean UI** — Minimal, modern interface built with pure HTML/CSS/JS. No bloat, no framework.
